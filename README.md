swagger-ui
===============================

自定义Venus-Boot风格的RESTful-API文档

```
  <dependency>
        <groupId>com.venus.boot.swagger</groupId>
        <artifactId>venus-boot-swagger-ui</artifactId>
        <version>1.0.0</version>
  </dependency>

```

支持如下8种主题样式

* 默认
* Feeling Blue
* Material
* Monokai
* Flattop
* Muted
* Newspaper
* Outline
